## Model Interface

```@docs
Optimisers.setup
Optimisers.update
Optimisers.update!
```

Calling `Functors.@functor` on your model's layer types by default causes the
optimiser to act on all suitable fields. To restrict this, define `trainable`:

```@docs
Optimisers.trainable
```
